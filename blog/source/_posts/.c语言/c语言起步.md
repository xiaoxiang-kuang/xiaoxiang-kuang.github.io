### 变量

* 变量定义：告诉编译器在何处创建变量，以及何处存储变量。
* 变量声明：向编译器保证变量以指定的类型和名称存在，这样编译器在不需要知道变量完整细节的情况下也能进一步的编译。

```c
extern int i;//声明，不是定义
int j;//声明，也是定义
```

* extern：表示变量是在别的位置定义的。

### 常量

* 整数常量可以是十进制、八进制（07）和十六进制（0xA1)，整数常量也可以带一个后缀，后缀是U（无符号数）和L的组合（long）。
* 浮点常量由整数部分、小数点、小数部分和指数部分组成
* `const int con = 1;`即可定义常量

### 存储类

* 定义C程序中变量、函数的范围及它们的生命周期。
  * auto是所有局部变量默认的存储类
  * register存储类用于定义存储在寄存器中的局部变量（取决于硬件和实现的限制）
  * static存储类：使用static修饰的局部变量可以保持局部变量的值。使用static修饰的全局变量会将变量的作用域限制在它的文件中
  * extern：用于提供一个全局变量、函数的引用,它是用来在另一个文件中声明一个全局变量或函数

### 运算符

* `&`与，`|`或，`^`异或，`~`取反

### 枚举

* 第一个枚举成员的默认值为整形的0，后续的成员的值在前一个成员上加1。

```c
enum DAY
{
    MON=1, TUE, WED, THU, FRI, SAT, SUN
};
enum DAY day;
```

### 指针

* 每一个变量都有一个内存位置，可以使用`&`来访问地址。

---

* ANSI C定义了许多宏，在编程中可以直接使用这些宏，但不能直接修改这些预定义宏
  * `__DATE__`：当前日期
  * `__TIME__`：当前时间
  * `__FILE__`：当前文件名
  * `__LINE__`：当前行号
