### 思想

小数上冒，大数沉底。

经过第一趟排序后，最大的数就移到了最后面。

第二趟排序会将次大值移到倒数第二位。

以此类推，直到所有的数排好位置。

### 代码实现

```java
    public void bubbleSort(int[] a){
        //比较n-1次
        for(int i=1;i<a.length;i++){
            for(int j=1;j<=a.length-i;j++){
                //两两进行比较
                if(a[j-1]>a[j]){
                    int temp=a[j];
                    a[j]=a[j-1];
                    a[j-1]=temp;
                }
            }

        }
    }
```

### 复杂度

时间复杂度：O(n²)

空间复杂度：O(1)

稳定的排序算法
