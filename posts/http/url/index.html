url参数（get） #

  • 每个参数都是键值对，用&分开
  • 字符a-z，A-Z，0-9，. - * _不变。
  • 用+替换空格。
  • 将其他所有字符编码为UTF-8，并将每个字节都编码为%后跟一个两位的16进制数字。
